## 安装 Flutter 的步骤

### 第一步：下载 Flutter SDK

打开终端（Terminal），依次执行以下命令：

```bash
# 1. 进入桌面
cd ~/Desktop

# 2. 下载 Flutter（这可能需要几分钟）
git clone https://github.com/flutter/flutter.git -b stable

# 3. 将 Flutter 添加到环境变量
echo 'export PATH="$PATH:$HOME/Desktop/flutter/bin"' >> ~/.zshrc

# 4. 重新加载环境变量
source ~/.zshrc

# 5. 验证安装
flutter --version
```

### 第二步：检查 Flutter 环境

```bash
flutter doctor
```

这个命令会检查您的系统是否满足 Flutter 运行的要求。

**如果显示有错误，请安装缺失的组件：**
- macOS 系统：确保安装了 Xcode（从 App Store 安装）
- Android 开发：可能需要安装 Android Studio

---

## 安装完成后，运行 App

```bash
# 1. 进入项目目录
cd ~/Desktop/expense_tracker_app

# 2. 安装项目依赖
flutter pub get

# 3. 运行应用
```

### 选择运行方式：

#### 方式 1：在 Chrome 浏览器中运行（最简单，推荐新手）

```bash
flutter run -d chrome
```

**优点：**
- 无需任何设备或模拟器
- 立即可以看到效果
- 不需要安装 Xcode 或 Android Studio

**缺点：**
- 不能使用相机拍照功能（其他功能都可以用）

#### 方式 2：在 iPhone 模拟器中运行

```bash
# 先启动模拟器
open -a Simulator

# 等待几秒后运行应用
flutter run
```

**优点：** 完整功能，包括相机和文件访问

#### 方式 3：在 Android 模拟器中运行

```bash
# 先启动 Android 模拟器（需要先安装 Android Studio）
flutter run -d android
```

## 常见问题

### 问题 1：git 命令找不到

**解决方案：** 安装 Git 或使用其他方式下载 Flutter

### 问题 2：flutter doctor 显示很多错误

**解决方案：**
- 如果只是想测试功能，可以忽略大部分错误
- 至少需要解决 "Flutter" 和 "Android toolchain" 或 "Xcode" 中的一个

### 问题 3：没有设备可以运行

**解决方案：** 使用 `flutter run -d chrome` 在浏览器中运行
