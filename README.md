# 太素记账

一个简单易用的在线记账应用，支持个人财务管理、自定义分类、数据导出等功能。

## 功能特点

- ✅ 记录收支：轻松记录每一笔收入和支出
- ✅ 自定义分类：支持自定义收入和支出分类
- ✅ 数据统计：实时显示总收入、总支出和余额
- ✅ 数据导出：支持导出 Excel (CSV) 和 PDF 格式
- ✅ 本地存储：数据保存在浏览器本地，隐私安全
- ✅ 响应式设计：支持电脑、平板和手机

## 使用方法

1. 访问应用：[在线地址](https://taisu-accounting.vercel.app/)
2. 添加记录：点击"添加"标签，填写金额、分类等信息
3. 查看记录：在"记录"标签查看所有账单
4. 导出数据：在"导出"标签选择时间范围并导出
5. 自定义分类：在"设置"标签管理您的分类

## 在手机上使用

### 方法一：添加到主屏幕（推荐）

1. 在手机浏览器中打开应用
2. 点击"分享"按钮
3. 选择"添加到主屏幕"
4. 现在可以像原生应用一样使用

### 方法二：直接使用

在任何现代浏览器中打开应用即可使用。

## 数据安全

- 所有数据保存在您浏览器的本地存储中
- 数据不会上传到任何服务器
- 清除浏览器数据会删除所有记录
- 建议定期导出数据备份

## 技术栈

- 纯 HTML5 + CSS3 + JavaScript
- LocalStorage API（本地数据存储）
- 响应式设计
- 无需后端服务器

## 部署

本应用部署在 Vercel 上，提供永久在线访问。
