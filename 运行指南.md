# 记账 App 运行指南

## 运行步骤

### 1. 安装 Flutter
打开终端，执行以下命令：

```bash
# 下载 Flutter SDK
cd ~/Desktop
git clone https://github.com/flutter/flutter.git -b stable

# 设置环境变量（添加到 ~/.zshrc 或 ~/.bash_profile）
export PATH="$PATH:$HOME/Desktop/flutter/bin"

# 重新加载配置
source ~/.zshrc
```

### 2. 验证 Flutter 安装
```bash
flutter doctor
```

### 3. 进入项目目录
```bash
cd ~/Desktop/expense_tracker_app
```

### 4. 安装依赖
```bash
flutter pub get
```

### 5. 运行应用
```bash
# 在模拟器上运行
flutter run

# 或者在真机上运行
flutter run -d <设备ID>
```

## 常见问题

### 问题 1: flutter 命令找不到
**解决方案：** 检查环境变量是否正确设置

### 问题 2: 没有连接设备
**解决方案：**
- 连接手机并开启 USB 调试
- 或者启动 Android 模拟器
- 或者使用 Chrome 浏览器运行

### 问题 3: 依赖安装失败
**解决方案：**
```bash
flutter clean
flutter pub get
```
