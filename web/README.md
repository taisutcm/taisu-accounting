# Web 版记账 App - 使用指南

## 如何使用

### 1. 在电脑上使用

**方法一：直接打开文件**
```bash
# 打开文件
open ~/Desktop/expense_tracker_app/web/index.html
```

**方法二：在浏览器中打开**
1. 打开 Finder（访达）
2. 导航到：桌面 → expense_tracker_app → web
3. 双击 `index.html` 文件
4. 文件会在默认浏览器中打开

---

### 2. 在 iPhone 上使用

**方法一：通过 Safari 浏览器**
1. 将 `index.html` 文件发送到 iPhone
   - 使用 AirDrop
   - 通过微信、邮件等方式
2. 在 iPhone Safari 中打开文件
3. 点击"分享"按钮 → "添加到主屏幕"

**方法二：通过本地服务器**
在 Mac 上运行本地服务器，然后通过 Safari 访问：

```bash
# 进入项目目录
cd ~/Desktop/expense_tracker_app/web

# 启动本地服务器（使用 Python）
python3 -m http.server 8000
```

然后在 iPhone Safari 中访问：`http://你的电脑IP地址:8000`

## 功能说明

### ✅ 已实现功能

1. **记录管理**
   - 添加收入/支出记录
   - 查看记录列表
   - 删除记录
   - 金额统计（总收入、总支出、余额）

2. **分类管理**
   - 支出分类：餐饮、交通、购物、娱乐、医疗、教育、住房、其他
   - 收入分类：工资、奖金、投资、兼职、其他

3. **日期管理**
   - 选择记账日期
   - 按时间排序显示

4. **备注功能**
   - 添加记录备注
   - 方便记录详细说明

5. **数据导出**
   - 导出为 CSV 文件（可在 Excel 中打开）
   - 导出为 PDF（通过打印功能）
   - 支持按时间范围筛选导出（本月、本季度、今年、全部）

6. **本地存储**
   - 数据保存在浏览器本地存储中
   - 刷新页面不会丢失数据
   - 关闭浏览器后数据依然保存

## 使用步骤

### 添加记录

1. 点击"添加"标签页
2. 选择类型：支出/收入
3. 输入金额
4. 选择分类
5. 选择日期（默认今天）
6. 添加备注（可选）
7. 点击"保存记录"

### 查看记录

1. 点击"记录"标签页
2. 查看所有记录列表
3. 查看统计信息（总收入、总支出、余额）
4. 点击"删除"按钮删除记录

### 导出数据

1. 点击"导出"标签页
2. 选择时间范围
3. 点击"导出 Excel"下载 CSV 文件
4. 点击"导出 PDF"打开打印窗口

## 在 iPhone 上使用提示

### 添加到主屏幕

1. 在 Safari 中打开记账 App
2. 点击底部的"分享"按钮（方框带向上箭头）
3. 滚动找到"添加到主屏幕"
4. 点击"添加"
5. 现在记账 App 就像原生应用一样在主屏幕了！

### 数据同步

⚠️ **重要提示**：
- 数据保存在浏览器本地存储中
- 不同浏览器（Safari、Chrome）之间数据不共享
- 清除浏览器数据会删除所有记录
- 建议定期导出数据备份

### 当前 Web 版本限制

❌ **暂不支持的功能**：
- 相机拍照（可以使用相册选择图片）

- 无需安装，直接使用
- 跨平台，支持所有浏览器

## 常见问题

### Q: 数据会丢失吗？
A: 数据保存在浏览器本地存储中，清除浏览器数据或使用隐身模式会导致数据丢失。建议定期导出数据备份。

### Q: 可以在多个设备间同步数据吗？
A: 当前版本不支持云端同步，需要通过导出/导入文件的方式在设备间传输数据。

### Q: 如何备份数据？
A: 点击"导出"标签页，选择时间范围，点击"导出 Excel"下载 CSV 文件作为备份。

### Q: iPhone 上如何使用？
A: 在 Safari 中打开文件，然后"添加到主屏幕"即可像原生应用一样使用。
