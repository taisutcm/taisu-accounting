# 在 iPhone 上直接使用记账 App

## 方法一：通过 App Store 安装（推荐，最简单）

### 步骤 1：构建 iOS 应用

我需要为您创建一个完整的 Flutter 项目，然后构建 iOS 应用。

### 步骤 2：分发方式选择

您可以选择：
- **TestFlight**：苹果官方测试平台，免费，可安装到多个设备
- **AltStore**：第三方签名工具，可以侧载应用
- **企业证书**：需要付费，可大规模分发

## 方法二：通过 Xcode 直接安装到 iPhone

### 前置条件
- 需要安装 Xcode（从 App Store 免费下载）
- 需要 Apple ID
- iPhone 和 Mac 在同一网络

### 步骤

```bash
# 1. 进入项目目录
cd ~/Desktop/expense_tracker_app

# 2. 安装依赖
flutter pub get

# 3. 确保 iPhone 已连接并信任电脑
# 打开 iPhone 的：设置 > 通用 > 关于本机 > 证书信任设置

# 4. 在 Xcode 中打开项目
open ios/Runner.xcworkspace

# 5. 在 Xcode 中：
# - 选择您的 iPhone 作为目标设备
# - 点击 Run 按钮（▶️）
# - 使用您的 Apple ID 签名
# - 安装应用到 iPhone
```
