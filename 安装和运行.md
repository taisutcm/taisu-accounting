# 记账 App - 安装和运行指南

## 第一步：检查 Flutter 是否已安装

打开终端（Terminal），输入：

```bash
flutter --version
```

**如果显示版本号**：说明 Flutter 已安装，直接跳到第三步

**如果显示 "command not found"**：需要先安装 Flutter

---

## 第二步：安装 Flutter

### 方法 1：使用 Git 克隆（推荐）

```bash
# 1. 下载 Flutter 到桌面
cd ~/Desktop
git clone https://github.com/flutter/flutter.git -b stable

# 2. 添加到环境变量
echo 'export PATH="$PATH:$HOME/Desktop/flutter/bin"' >> ~/.zshrc

# 3. 重新加载配置
source ~/.zshrc

# 4. 验证安装
flutter --version
```

### 方法 2：手动下载

1. 访问：https://docs.flutter.dev/get-started/install/macos
2. 下载 Flutter SDK
3. 解压到桌面
4. 执行上面的环境变量设置命令

---

## 第三步：配置 Flutter 环境

```bash
# 检查 Flutter 环境状态
flutter doctor
```

按照提示安装缺失的依赖（如 Xcode、Android Studio 等）

---

## 第四步：运行记账 App

```bash
# 1. 进入项目目录
cd ~/Desktop/expense_tracker_app

# 2. 安装项目依赖
flutter pub get

# 3. 运行应用（选择其中一种方式）
```

### 运行方式选择：

#### 方式 1：在 Chrome 浏览器中运行（最简单）
```bash
flutter run -d chrome
```
**优点**：无需任何设备，直接在浏览器中运行
**缺点**：不能使用相机功能

#### 方式 2：在 iOS 模拟器中运行
```bash
# 先启动模拟器
open -a Simulator

# 然后运行应用
flutter run
```
**优点**：完整功能体验
**缺点**：需要 Xcode

#### 方式 3：在真机上运行

**iPhone：**
```bash
flutter run -d <你的设备ID>
```

**Android：**
```bash
flutter run -d <你的设备ID>
```

## 常见问题

### 问题 1：flutter pub get 失败
```bash
flutter clean
flutter pub get
```

### 问题 2：没有找到设备
```bash
# 查看可用设备
flutter devices

# 启动 Chrome 作为目标设备
flutter run -d chrome
```

### 问题 3：权限错误
确保您有读写权限，可以在项目目录执行：
```bash
sudo chown -R $(whoami) .
```
