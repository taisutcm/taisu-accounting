# 记账 App - 运行指南

## 情况说明

您的系统上还没有安装 Flutter，需要先安装 Flutter 才能运行这个 App。

## 推荐方案

### 方案 1：安装 Flutter（推荐，功能最完整）

#### 步骤 1：下载 Flutter SDK
打开终端（Terminal），执行：

```bash
# 1. 进入桌面
cd ~/Desktop

# 2. 下载 Flutter
git clone https://github.com/flutter/flutter.git -b stable

# 3. 将 Flutter 添加到环境变量
echo 'export PATH="$PATH:$HOME/Desktop/flutter/bin"' >> ~/.zshrc

# 4. 重新加载环境变量
source ~/.zshrc

# 5. 验证安装
flutter --version
```

#### 步骤 2：检查 Flutter 环境
```bash
flutter doctor
```

这会显示需要安装的依赖项（如 Xcode、Android Studio 等）。

#### 步骤 3：安装依赖并运行
```bash
# 进入项目目录
cd ~/Desktop/expense_tracker_app

# 安装依赖
flutter pub get

# 运行应用（选择其中一个）
# 选项 A：在 Chrome 浏览器中运行（最快）
flutter run -d chrome

# 选项 B：在模拟器中运行
# 先启动模拟器：open -a Simulator
flutter run

# 选项 C：在真机运行
flutter run -d <设备ID>
```
